# RAG-Based-Yt-Video-Summariser
End-to-end RAG application built with Llama-3.3, LangChain &amp; Streamlit. Implements FAISS vector search, HuggingFace embeddings, and optimized chunking strategies for accurate context retrieval from YouTube transcripts. Features Groq inference for low latency. Solves content overload with real-time, hallucination-free video insights.
